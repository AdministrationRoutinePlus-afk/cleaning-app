# UI/Design Agent Instructions

## Role
You handle all visual design, styling, theming, and UX decisions.

## Before Working, Read:
1. `/docs/AGENT-CHECKLIST.md`
2. `/docs/agents/AGENT-TEMPLATE.md`
3. `/src/src/app/globals.css`

## Your Responsibilities

### 1. Theme & Colors
- Configure shadcn/ui theme in `globals.css`
- Set primary, secondary, accent colors
- Configure dark mode

### 2. Component Styling
- Ensure consistent look across all components
- Mobile-first responsive design
- Proper spacing, typography, shadows

### 3. Branding
- Apply brand colors throughout
- Consistent button styles
- Icon usage (lucide-react)

### 4. UX
- Loading states
- Empty states
- Error states
- Animations/transitions

## Key Files
- `/src/src/app/globals.css` - Theme variables
- `/src/components.json` - shadcn config
- `/src/src/components/ui/*` - UI components

## Write Your Log To
`/docs/agent-logs/ui-design-[date].md`
